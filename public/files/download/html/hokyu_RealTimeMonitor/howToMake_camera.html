1.カメラ本体 アプリからカメラを制御する為に SONYのRemote API に対応したカメラ が必要です。WEBサイトに記載は無い DSC-WX500 (2017年当時¥33,000)で動作しました。カメラのアプリケーション→アプリ一覧→ スマートリモコン内蔵版 があれば動作すると思います。貴社のSONYカメラでトライしてみてください。もし新しく購入する場合は焦点距離が50mm程度が実物に近い写真が撮れるのでお勧めです。

2.LEDテープ WS2812Bを使用。必要な長さは60cm。100cmで¥2,000程度。Arduinoに接続して照明の色を変更します。コネクタ色によって照明色を変更しないと上手く撮影出来ません。より実物に近い色で撮影しないと直感的に どのハメ図か分からなかったので追加しました。これが不要な場合はArduinoも必要ありません。LEDタイプには複数の種類があります。より新しいLEDテープは抵抗が少なく照明が明るすぎる可能性がありますが特に問題ありません。

3.ArduinoArduinoは組込み済みマイコンボード。LEDテープの色と明るさを制御する為に使用しました。最も有名な電子工作用の開発ボードでWEB上で情報が豊富。 3Dプリンターの制御にもよく使用されます。ここでは最も有名なArduino UNO(¥3,000)を使用。互換品のArduinoでも動作します。

4.無線LAN子機 カメラとの無線通信で使用(約¥3,000-7,000)PCに接続して使用します。カメラとの無線通信は通常のネットワークとは別に接続する必要があります。 ※使用PCがノートパソコンで有線LANのみ使用している場合のみ不要です。

ArduinoIDEのインストール
ArduinoIDE(統合開発環境)はPCからArduinoにプログラムをインストールする為のソフト

組込み済みマイコンボード
最も有名な電子工作用の開発ボードで情報が豊富
3Dプリンターの制御にもよく使用されます
ここでは最も有名なArduino UNO(¥3,000)を使用
より最適な照明でコネクタ撮影をする為に利用
自分はATmega328P系を使用しましたが、安価な互換品Arduinoでも動作します

Arduinoにプログラムを書き込む

1.PCとArduinoをUSBケーブルで接続します

この時PC側USBポートは給電専用を使用しないで下さい
給電専用の場合は ←こんな感じのマークが印刷されています

2.プログラムのダウンロード

下記からダウンロードして展開(解凍)して下さい
Arduino用のコード

3.保存をクリック

※Windowsのバージョンによって画面が異なるかもしまれません

4.フォルダーを開くをクリック

5.選択されているファイルを右クリック、すべて展開をクリック

6.展開をクリック

7.展開されたフォルダが表示されるので開く

8.001_satuei_shomei_**を開く

ArduinoIDEがインストール済みの場合、ダブルクリックで開きます

9.ツール→シリアルポートの順にクリックして、Arduinoとして認識されているCOMを選択

ここではCOM4となっていますが、この番号はPCによって異なります

10.ツール→ボードの順にクリックして、Arduino/Genuino Unoを選択

11.ツール→ライブラリを管理の順にクリック

12.検索欄に Adafruit_neopixel と入力

13.検索結果が複数表示されるのでAdafruit NeoPixel by Adafruit のバージョン1.1.8 を選択して、インストールをクリック

14.インストール後、タイプの一覧からインストール済みを選択して、先ほどインストールしたものが表示されていたら成功

15.右矢印のボタンを押すと、コンパイルして書き込みが開始されます

16.ボードへの書き込みが完了しました。が表示されたらコードの書き込み完了です

失敗する場合は、ツールの設定などこの説明通りに選択されているか確認してください
PCの通信設定
※カメラの固定IPと通信する為の設定
1.ウィンドウズキーを押す
　そのまま「インターネット」 と入力
　インターネットオプションをクリック

2.[接続] → [LANの設定]の順にクリック

3.[詳細設定]をクリック

4.例外の末尾に「;192.168.122.1」を追記

これでカメラとPCの無線通信設定が完了しました
PCとカメラの通信設定 1.カメラの電源を入れ▶ボタンを押すと、下の図の様に表示されるので実行を選択

2.MENUボタンを押してアプリ一覧を選択する

3.スマートリモコン内臓版を選択する

4.PCのWIFI接続からカメラに表示されたSSIDを選択
　パスワードを入力して接続する

5.接続中の時にCAMERA+アプリのRecModeを押すと接続できます
